* Structure and Union             Structure = The defination of a structure informs the 'c' compiler about the type and the number of data members froming the structure.Members can be accessed by (.) operator. A structure can be defined in a c language the key word struct followd by it's name and a body enclosed ina curly braces. The body of the structure's contain the defination of its members and each membershave same name.declaration ends by semicolon;.
syntax:
struct[structure_tag]
{
member defination 1;
member defination 2;
member defination 3;
.....
member defination n;
}
[one or more structure variable.]

b)Union : Union is a special data type avaliable in c .We use keyword for defining a structure.In union all members are allocated in the same memory address.Variables in union can be declare at the time of defination or after union type defination in seperate declaration statment.Members can be accessed by (.) operator.

syntax:
union[union name]
{
member defination 1;
member defination 2;
member defination 3;
.....
member defination n;
}
[one or more union variable.]
