#include stdio.h
void main() {
int color = 1;
printf("please choose a color \n");
printf("1:red\t 2:green \t 3:blue");
scanf("%d", &color);
switch(color)
{
case 1: printf("you choose a red color\n");
break;
case 2: printf("you choose a green color\n");
break;
case 3: printf("you choose a blue color\n");
break;
default:
printf("you did not choose any color");
}
}
*Code Explaination*
Note= *stdio.h* Will be in a angular bracket otherwise code not run.
  • #include = The #include directive tells the C preprocessor to include
    the contents of the file specified in the input stream to the compiler
    and then continue with the rest of the original file. ... A header file may contain any valid C program fragment.
  • stdio.h = stdio.h is a header file in C, it is the file which contains C declaration and Macro definition to be shared between several files. stdio.h means standard input/output function which contains printf(), scanf() functions.
  • int = int is a data type which contain integer type of data.
  • main() = main() is not a predefined or inbuilt function. It is a user-defined function with a predefined function prototype (also called function signature). The user writes its functionality, but its declaration has certain restrictions.
  • printf() or scanf() = "printf" is the name of one of the main C output functions, and stands for "print formatted". printf format strings are complementary to scanf format strings, which provide formatted input (parsing). ... The format string itself is very often a string literal, which allows static analysis of the function call.scanf take input from user.
  • return = return 0 in C Programming Language: return value of the main function is considered the "Exit Status" of the application. On most operating systems returning 0 is a success status like saying "The program worked fine".

  • switch() = switch statment checksthe value of a expression against a case values,if condition matches the case value then control transferred to that point if condition not match then default will execute.
