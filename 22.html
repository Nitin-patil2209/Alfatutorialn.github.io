* Data type's             C language supports 2 different type of data types:

            1)Primary data types: These are fundamental data types in C namely integer(int), floating point(float), character(char) and void.
a)int = Integers are used to store whole numbers. int or signed int   bytes 2   range-32,768 to 32767
unsigned int   bytes 2   range 0 to 65535
short int or signed short int   bytes 1   range -128 to 127
unsigned short int   bytes 1   range 0 to 255
long int or signed long int   bytes 4   range -2,147,483,648 to 2,147,483,647
unsigned long int   bytes4   range 0 to 4,294,967,295

b)float = Floating types are used to store real numbers.
Float   size 4   range 3.4E-38 to 3.4E+38
double   size 8   range 1.7E-308 to 1.7E+308
long double   size 10   range 3.4E-4932 to 1.1E+4932

c) char = Character types are used to store characters value.size of char is 1.;
void type d) void type means no value. This is usually used to specify the type of functions which returns nothing. We will get acquainted to this datatype as we start learning more advanced topics in C language, like functions, pointers etc.
            2)Derived data types: Derived data types are nothing but primary datatypes but a little twisted or grouped together like array, stucture, union and pointer. These are discussed in details later.
