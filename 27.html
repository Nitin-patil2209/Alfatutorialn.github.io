* Conditional Statment             : A conditional statement, symbolized by p q, is an if-then statement in which p is a hypothesis and q is a conclusion. The logical connector in a conditional statement is denoted by the symbol . The conditional is defined to be true unless a true hypothesis leads to a false conclusion. Conditional statements help you to make a decision based on certain conditions. These conditions are specified by a set of conditional statements having boolean expressions which are evaluated to a boolean value true or false.
Types of conditional statment.
a)if = The statements inside the body of “if” only execute if the given condition returns true. If the condition returns false then the statements inside “if” are skipped.
syntax :
if (condition) {
//Block of C statements here
//These statements will only execute if the condition is true
}

b)if else = The if-else statement in C is used to perform the operations based on some specific condition.In if else statment allows us to select one of the two variables depending of the output of the expression if statment is true then if part of program will exicuted other wise else part exicuted.
syntax :
if(condition)
{
true statment;
}
else
{
false statment;
}

c) switch = switch statment is a multiple or multiway branching decision making statment.Switch statment checks the value of a expression against a case value, if condition matches the case value then control is transfer to that point.
syntax:
switch(expression)
{
case expr1:statment;
break;
case expr2:statment;
break;
case expr3:statment;
break;
case expr4:statment;
break;
}
